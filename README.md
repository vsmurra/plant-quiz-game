# plant-quiz-game

Logo: PlantsQuizGame

# Test your plant knowledge in this quiz game! Answer correctly to win. You only have 3 lives.

# How to Play
- Click "Start Game" to begin.
- Choose the correct answer to each question.
- If you lose all your lives, the game ends!
